# 2025-01-15 - 《伤寒论再发掘》第十九章 一般的伤寒论 15-16

# **19 ·（15）**

# **222. 伤寒“二三日”心中悸而烦者“小”建中汤主之（107）**

此条文在“原始伤寒论”中是第29条（参见第18章10项），在《伤寒论讲义》中为第107条，但在形成“初期伤寒论”时有所改动。即增加了关于“二三日”的解释句和“小”字。其余部分与“原始伤寒论”完全相同。

# **223. 太阳病“过经十余日”反二三下之后“四五日柴胡证仍在者先与小柴胡汤”呕不止心下急郁郁微烦者“为未解也与”大柴胡汤主之\*1（108）【方**】

**柴胡半斤 黄芩三两 半夏半斤 生姜五两 芍药三两 枳实四枚炙 大枣十二枚擘\*2**

**右七味以水一斗二升煮取六升去滓再煎取三升温服一升日三服 “一方加大黄二两若不加恐不为大柴胡汤”**

此条文在“原始伤寒论”中是第30条（参见第18章10项），在《伤寒论讲义》中为第108条，但在形成“初期伤寒论”时有较大改动。除了增加了关于“过经十余日”的解释句外，还添加了“四五日柴胡证……”和“为未解也……”等解释句。宋板中\*1处为“下之则愈”，显得像是解释句。此外，\*2处的生药配列稍显混乱，已正。最后的“一方加大黄……”显然是后人旁注，应归入“……”。

通过这条文可以看出，“原始伤寒论”不可能仅通过宋板的简单摘录来创建。

# **224. “伤寒十三日不解胸胁满而呕日晡所发潮热已而微利此本柴胡证下之以不得利今反利者知医以丸药下之此非其治也潮热者实也先宜服小柴胡汤以解外后以柴胡加芒硝主之”（109）【方**】

**“柴胡二两十六铢 黄芩一两 人参一两 甘草一两炙 生姜一两切 半夏二十铢 本云五枚洗 大枣四枚擘 芒硝二两”**

**“右八味以水四升煮取二升去滓内芒硝更煮微沸分温再服 不解更作”**

此条文在《伤寒论讲义》中为第109条，但不是“原始条文”，而是“初期条文”。因为“柴胡加芒硝汤”的生药组合虽基于“小柴胡汤”加上“芒硝”，但生药配列和用量完全不同，且使用了“铢”作为计量单位，明显是“原始伤寒论”之后时代的条文。

# **225. “伤寒十三日过经谵语者以有热也当以汤下之若小便利者大便当硬而反下利脉调和者知医以丸药下之非其治也若自下利者脉当微厥今反和者此为内实也调胃承气汤主之”（110）**

此条文在《伤寒论讲义》中为第110条，但不是“原始条文”，而是“初期条文”。因为其中出现了“丸药”和“医”的描述，显然属于“原始伤寒论”之后的时代。调胃承气汤的使用方法实际，因此可以视为“初期条文”。

# **226. 太阳病“不解”热结膀胱其人如狂血自下下者愈“其外不解者尚未可攻当先解其外外解已”但少腹急结者“乃可攻之”与\*1桃仁承气汤（111）【方】**

**“桃仁五十个去皮尖 大黄四两酒洗 甘草二两炙 芒硝二合 桂枝二两去皮\*2”**

**右五味以水七升煮取二升半去滓内芒硝更上微火一两沸\*3温服五合**

此条文在“原始伤寒论”中是第31条（参见第18章10项），在《伤寒论讲义》中为第111条，但在形成“初期伤寒论”时有较大改动。宋板中增加了“不解”、“其外……”、“乃可……”等“初期条文”，\*1处的“与”变为“宜”，\*2处的生药配列为（桃仁•大黄•桂枝•甘草•芒硝），汤剂形成过程变得不明。\*3处增加了更多解释元素。这些改变动反映了后来时代的改动。汤名中的“桃核”应为“桃仁”。

# **227. “伤寒八九日下之胸满烦惊小便不利谵语一身尽重不可转侧者柴胡加龙骨牡蛎汤主之”（112）【方】**

**“柴胡四两 龙骨 黄芩 生姜切 铅丹 人参 桂枝去皮 茯苓各一两半 半夏二合半洗 大黄二两 牡蛎一两半熬 大枣六枚擘”**

**右十二味以水八升煮取四升内大黄切如棋子更煮一两沸 去滓温服一升“本云柴胡汤今加龙骨等”**

此条文在《伤寒论讲义》中为第112条，但不是“原始条文”，而是“初期条文”。因

为汤名、生药配列完全没有规律性，且生药用量记述方式如“各一两半”也是“原始伤寒论”时代没有的后期表达。

“铢”作为计量单位也出现在这里，这是“原始伤寒论”中所没有的，因此这显然是后期的条文。

# **228. （伤寒腹满谵语寸口脉浮而紧此肝乘脾也名曰纵刺期门）（113）**

# **229. （伤寒发热恶寒大渴欲饮水其腹必满自汗出小便利其病欲解此肝乘肺也名曰横刺期门）（114）**

《伤寒论讲义》中说明以上两条不被视为正文内容（第113条、第114条）。由于提到“肝乘脾”和“肝乘肺”，明显受到“五行说”的影响，因此这些显然是“初期伤寒论”中的后人旁注，在后来的“后期伤寒论”中被提升为条文，即视为“后期条文”。

# **230. （太阳病二日反躁凡熨其背而大汗出大热入胃胃中水竭躁烦必发谵语十余日振慄自下利者此为欲解也故其汗从腰以下不得汗欲小便不得反呕欲失溲足下恶风大便硬小便当数而反不数及不多大便已头卓然而痛其人足心必热谷气下流故也）（115）**

# **231. （太阳病中风以火劫发汗邪风被火热血气流溢失其常度两阳相熏灼其身发黄阳盛则欲衂阴虚小便难阴阳倶虚竭身体则枯槁但头汗出剂颈而还腹满微喘口干咽烂或不大便久则谵语甚者至哕手足躁扰捻衣摸床小便利者其人可治）（116）**

以上两条在《伤寒论讲义》中不被视为正文内容（第115条、第116条）。含有“谷气”、“阳盛”或“阴虚”等术语的条文很难视为“原始条文”或“初期条文”，最终应视为“后期条文”。

# **232. “伤寒脉浮医以火迫劫之亡阳必惊狂卧起不安者桂枝去芍药加蜀漆牡蛎龙骨救逆汤主之”（117）【方】**

**“桂枝三两去皮 甘草二两炙 生姜三两切 大枣十二枚擘 牡蛎五两熬 蜀漆三两洗去腥 龙骨四两”**

**“右七味以水一斗二升先煮蜀漆减二升内诸药煮取三升去滓温服一升 本云桂枝汤今去芍药加蜀漆牡蛎龙骨”**

此条文在《伤寒论讲义》中为第117条，但不是“原始条文”，而是“初期条文”。因为出现了“医”的描述，表明它是医官制度确立后的条文，推测为“初期伤寒论”时代的作品。

# **233. （形作伤寒其脉不弦紧而弱弱者必渴被火必谵语弱者发热脉浮解之当汗出愈）（118）**

# **234. （太阳病以火熏不得汗其人必躁到经不解必清血名为火邪）（119）**

# **235. （脉浮热甚而反灸之此为实实以虚治因火而动必咽燥吐血）（120）**

# **236. （微数之脉慎不可灸因火为邪则为烦逆追虚逐实血散脉中火气虽微内攻有力焦骨伤筋血难复也）（121）**

# **237. （脉浮宜以汗解用火灸之邪无从出因火而盛病从腰以下必重而痹名火逆也欲自解者必当先烦烦乃有汗而解何以知之脉浮故知汗出解）（122）（123）**

《伤寒论讲义》中说明以上五条（最后一条分为两条，共六条）不被视为正文内容（第118-第123）。这些条文主要涉及字句解释或理论探讨，与实际治疗几乎没有关系，因此更适合视为“后期条文”，而不是“初期条文”。在这一部分，大量的“后期条文”已经被嵌入进来。

# **238. “烧针令其汗针处被寒核起而赤者必发奔豚气从少腹上冲心者灸其核上各一壮与桂枝加桂汤更加桂二两也”（124）【方】**

**“桂枝五两去皮 芍药三两 生姜三两切 甘草二两炙 大枣十二枚擘”**

**“右五味以水七升煮取三升去滓温服一升本云桂枝汤今加桂满五两所以如桂者以能泄奔豚气也”**

《伤寒论讲义》中为第124条。在“原始伤寒论”的时代还没有针灸的实际并用，因此文中完全没有提到“针灸”。更不用说使用“烧针”的经验了，所以这条文显然是更晚时代的产物。这是笔者认为它是“初期条文”的原因之一。另一个原因是生药配列不正确，正确的顺序应该是（桂枝•芍药•甘草•生姜•大枣）。

# **239. “火逆下之因烧针烦躁者桂枝甘草龙骨牡蛎汤主之”（125）【方】**

**“桂枝一两去皮 甘草二两炙 牡蛎二两熬 龙骨二两”**

**“右四味以水五升煮取二升半去滓温服八合 日三服”**

《伤寒论讲义》中为第125条。由于前一条文提到了“烧针”，这里记录了关于“烧针”的另一个汤剂经验，因此这也不是“原始条文”，而是推测为“初期条文”。此外，汤名（桂枝甘草龙骨牡蛎汤）和生药配列（桂枝•甘草•牡蛎•龙骨）也不对应，这一点也表明这不是“原始条文”，而是“初期条文”。

# **240. （太阳伤寒者加温针必惊也）（126）**

# **241. （太阳病当恶寒发热今自汗出反不恶寒发热关上脉细数者以医吐之过也一二日吐之者腹中饥口不能食三四日吐之者不喜糜粥欲食冷食朝食暮吐以医吐之所致也此为小逆）（127）**

# **242. （太阳病吐之但太阳病当恶寒今反不恶寒不欲近衣此为吐之内烦也）（128）**

# **243. （病人脉数数为热当消谷引食而反吐者此以发汗令阳气微隔气虚脉乃数也数为客热不能消谷以胃中虚冷故吐也）（129）**

《伤寒论讲义》中说明以上四条不被视为正文内容（第126-第129条）。笔者也认为这些是“初期伤寒论”中的后人旁注，在后来的“后期伤寒论”中被提升为条文。因为内容几乎都是与治疗无关的解释性文字，即属于“后期条文”的一种，因此用“……”括起来。

# **244. “太阳病过经十余日心下温温欲吐而胸中痛大便反溏腹微满郁郁微烦先此时自极吐下者与调胃承气汤若不然不可与但欲呕胸中痛微溏者此非柴胡汤证以呕故知极吐下也”（130）**

《伤寒论讲义》中为第130条，但这显然不是“原始条文”，而是“初期条文”。因为其中增加了“过经十余日”、“非柴胡汤证”等解释元素，这些更像是后人的旁注。

# **245. “太阳病六七日表证仍在脉微而沉反不结胸其人发狂者以热在下焦少腹当硬满小便自利者下血乃愈所以然者以太阳随经痛热在里故也抵当汤主之”（131）【方】**

**“水蛭熬 虻虫各三十个去翅足熬 桃仁二十个去皮尖 大黄三两酒洗”**

**“右四味以水五升煮取三升去滓温服一升不下更服”**

《伤寒论讲义》中作为正文内容进行了说明（第131条），但这也显然不是“原始条文”，而是“初期条文”。因为含有“六七日”、“以太阳随经”等字句，且汤名与生药配列无关，还有直接使用动物药材（而在“原始伤寒论”中只有蛋黄、牡蛎和阿胶），因此不具备“原始条文”的资格。这是将其视为“初期条文”的理由。

# **246. “太阳病身黄脉沈结少腹硬小便不利者为无血也小便自利者其人如狂者血证谛也抵当汤主之”（132）**

《伤寒论讲义》中为第132条，基于同样的理由，这也不是“原始条文”，而是“初期条文”。

# **247. “太阳病有热少腹满应小便不利今反利者为有血也当下之不可余药宜抵当丸”（133）【方】**

**"水蛭二十个 虻虫二十个去翅足熬 桃仁二十五个去皮尖 大黄三两"**

**“右四味捣分四丸以水一升煮一丸取七合服之昼夜当下血若不下者更服”**

第245条和第246条等让人联想到“原始伤寒论”的第34条（桃仁承气汤的条文，“宋板”中为第226条）。“血证”这一概念很可能是在“原始伤寒论”之后的时代产生的。

“抵当汤”这一名称的由来不明，但笔者根据以下推测认为（出自《汉方研究》1992年9月号，通卷249号，第26页）：“抵当”的意思除了“抵押”或“担保”之外，还有“作为紧急情况下的预备品”的意思。因此，“抵当汤”可能是指在使用其他汤剂（如桃仁承气汤）效果不佳时使用的备用汤剂，即在难以改善瘀血病态时使用的特别汤剂。

随着“抵当汤”的使用，后来可能为了方便制作丸药，产生了制作“抵当丸”的需求，因此关于“抵当丸”的条文也随之被创作出来。

# **248. （太阳病小便利者以饮水多必心下悸小便少者必苦里急也）（134）**

《伤寒论讲义》中说明这条文不是正文内容（第134条）。因为内容几乎与实际治疗无关，所以更适合视为“后期条文”，而不是“初期条文”。

在此条文中，“宋板”标志着“太阳病中篇”的结束，接下来进入“太阳病下篇”。本项中讨论了十一条文，其中没有一条是“原始条文”，六条为“初期条文”，五条为“后期条文”。

“初期条文”的特点是与实际治疗有一定关联的内容，而“后期条文”则主要是与实际治疗几乎没有关系的内容。

从“原始伤寒论”到“初期伤寒论”，再到“后期伤寒论”，随着时间推移，解释元素逐渐增加，这一点在上述条文中得到了具体体现。

在《伤寒论再发掘》的论述中，之所以特意讨论一般的“伤寒论”，是为了具体了解“原始伤寒论”在其结构中的重要性。因此，没有必要全文列出与之关系不大的“后期条文”，这样做既浪费精力又无意义。因此，今后对于“后期条文”，除了条文编号和开头部分外，其余内容将省略；对于“初期条文”，也将尽可能简化。

无论何种条文，阅读方式不同会有不同的意义，但笔者不喜欢保留“劳而无功”的内容。应当“保留好的，摒弃坏的”，以便后继者能够更好地继承和发展。
